# EMO-Drive

Real-time driver emotional state monitoring system built with Arduino and Python. Monitors heart rate, body temperature, eye blink rate, and gas levels to classify the driver's emotional state — Calm, Drowsy, Angry, Anxious, or Alcohol Detected — and streams live data to a web dashboard. At the end of a 10-minute session, an AI-generated safety report is produced with a driving score, fitness verdict, and personalised recommendations grounded in the Indian Motor Vehicles Act 1988.

Built with Flask, Arduino, and the NVIDIA NIM API.
